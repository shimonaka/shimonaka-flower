# DESIGN.md — 下仲生花店 公式サイト

## ブランド前提
- **店名**: 下仲生花店（しもなかせいかてん）
- **所在地**: 大阪府岸和田市加守町1-1-7
- **業態**: 街の生花店（地域密着・家族経営）
- **HPの位置付け**: 信頼担保の「存在証明」。リード獲得ツールではない
- **CTA**: 電話・来店のみ（注文フォーム禁止）

## デザインコンセプト
**クラフト温かみ系（案B）**

岸和田の街の花屋として、年配のお客様も若い世代も入りやすい中庸を狙う。
インスタの「いつもご贔屓に」「ありがとうございました」という人情系トーンと連続性を保つ。
洗練しすぎてチェーン店化しない／和風に振りすぎて弔事専門に見えない、その中間。

## タイポグラフィ

| 用途 | フォント |
|---|---|
| 見出し | `"Zen Maru Gothic", "Hiragino Maru Gothic ProN", sans-serif` (weight 500-700) |
| 本文 | `"Zen Maru Gothic", "Hiragino Sans", sans-serif` (weight 400) |
| UI（ナビ・小見出しラベル） | `"Inter", "Helvetica Neue", system-ui, sans-serif` (weight 400-500) |

**選定理由**: Zen Maru Gothic は丸みのある人情系ゴシック。明朝だと弔事寄り、Noto Sansだと無機質になる。Maruで温かみを出す。

### サイズスケール
- h1: `clamp(2.2rem, 5vw, 3.8rem)`（ヒーロー特大）
- h2: `clamp(1.6rem, 3.4vw, 2.4rem)`
- h3: `clamp(1.15rem, 2vw, 1.4rem)`
- 本文: `1rem` / 行間 `1.85`（日本語用にゆったり）

## 余白スケール
4 / 8 / 16 / 24 / 32 / 48 / 64 / 96 / 128 px（CSS変数 `--space-1〜9`）

## レイアウト
- コンテナ最大幅: 1120px
- セクション縦パディング: 64px〜96px（モバイル / デスクトップ）
- ヒーロー: `min-height: calc(100vh - var(--header-h))` 全画面表示

## ボタン
- 角丸: `--radius: 4px`（クラフト感を出すため僅かに丸める。シャープでもなく完全な丸でもない）
- ホバー: 背景透過 → ボーダー反転（地味だが品が出る）
- アンダーライン: アクセントカラーで `2px` 細線

## ナビゲーション
- 英字（WORKS / OCCASIONS / ACCESS / CONCEPT）
- letter-spacing 広めで余裕感
- カレントページは下線（accent color）

## 5ページ構成

| URL | NAV | 中身 |
|---|---|---|
| `/` | TOP | ヒーロー（薔薇ハート）／概要／ギャラリー抜粋／注文の案内 |
| `/works/` | WORKS | 全43枚をシーン別に分類して掲載 |
| `/occasions/` | OCCASIONS | 用途別の説明（祝・弔・装花・鉢物） |
| `/access/` | ACCESS | 地図埋込・営業時間・電話 |
| `/concept/` | CONCEPT | 店の想い（家族からの追記前提のドラフト） |

## strict mode 適用方針
- インスタキャプションに含まれる事実のみ並べる
- 「想いを込めて」「お客様に寄り添う」のような汎用美辞麗句は使わない
- 取扱領域の列挙はキャプションに登場した語句に基づく
- concept ページは家族からの追記を促すドラフトとして作成

## 禁止事項
- 注文フォーム・問い合わせフォーム
- 配達・宅配の言及（運用上のリスク回避）
- 価格表示
- スタッフ顔写真（後追い対応）
- 「AI」「自動」「ツール」など制作手段の言及
- 「現代の」「進化する」など流行語
